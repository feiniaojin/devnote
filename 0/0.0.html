开发笔记 #

项目介绍 #

本文档是笔者日常开发整理的笔记。

阅读方法 #

在线阅读 #

本文档已部署到Github Pages，请点击在线阅读。

本地阅读 #

本文档是由VitePress驱动的，可以通过克隆本工程进行本地编译构建运行。过程如下：

shell
# 克隆获取源码
$ git clone https://github.com/feiniaojin/kaifa.git

# 进入项目文件夹
$ cd kaifa

# 安装依赖
$ yarn add all

# 启动工程
$ yarn docs:dev

工程启动后，访问链接为:

shell
http://localhost:5173/
